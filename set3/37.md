# Oefening 37

Construeer een notification query die een event genereert telkens in een specifieke directory (bijvoorbeeld c:\temp) een bestand gecreÃ«erd wordt. Los deze vraag op twee manieren op, telkens vertrekkend van een andere WMI klasse.

Eerste manier:

```
SELECT * 
FROM __InstanceCreationEvent Within 10
WHERE TargetInstance ISA 'CIM_DirectoryContainsFile'
	AND TargetInstance.GroupComponent = "Win32_Directory.Name='c:\\temp'"
```

Tweede manier: een bestand staat in een bepaalde map als de drive en het pad overeenkomen met de directory

```
SELECT *
FROM __InstanceCreationEvent Within 10
WHERE TargetInstance ISA 'CIM_DataFile'
	AND TargetInstance.Drive = 'c:'
	AND TargetInstance.Path = "\\temp\\"

```
